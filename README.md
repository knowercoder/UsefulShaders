

# Useful Shaders for Unity

List of Unity shaders that are useful for your projects as well as for learning purposes.
- Shaders are created with Shader graph
- Supports URP
- Unity Version used 2022.3 LTS



## Shaders List

- Highlight
- Vertex Animation
- Camera Distance Mask
- Normal Blend
- Sizzling Dissolve
- WindShake
- Ripple
